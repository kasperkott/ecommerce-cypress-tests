# ecommerce-cypress-tests

UI and API test automation suite for a demo e-commerce application using Cypress.

## ✅ Features

- End-to-end UI testing (login, add to cart)
- REST API testing with `cy.request()`
- Page Object Model (POM) structure
- Custom Cypress commands
- Test data from fixtures
- Ready to use with `npx cypress open` or `npx cypress run`
